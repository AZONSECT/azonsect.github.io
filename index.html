Geleceğin Teknolojilerini Tasarlamak.

Elektronik, aviyonik ve gömülü sistemler alanında uzmanlaşan; roket, batarya teknolojisi, uydu görüntüleme ve çip tasarımı gibi yüksek teknolojili projelerde aktif görev almış mühendis adayı. FPGA (VHDL/Verilog), PCB tasarımı, mikrodenetleyici programlama ve yazılım geliştirme konusunda deneyim sahibi. Ar-Ge, savunma sanayi, enerji ve uzay teknolojileri alanlarında çalışmayı hedefleyen disiplinli, meraklı ve çözüm odaklı bir elektronik mühendisi.

Hakkımda

2001, Bakü doğumluyum. 2020 yılından bu yana Gebze Teknik Üniversitesi'nde Elektronik Mühendisliği lisans eğitimi almaktayım.

Eğitim hayatım boyunca roket teknolojileri, batarya sistemleri, uydu görüntüleme ve çip tasarımı gibi çeşitli AR-GE projelerinde aktif rol aldım. Teknofest gibi yarışmalarda finalist olma başarısı gösterdim. Problem çözmeye, ekip çalışmasına ve proje yönetimine odaklanarak kendimi sürekli geliştiriyorum.

Teknik Yetkinliklerim

C / C++ Java Python VHDL / Verilog FPGA Arduino Kicad (PCB Tasarımı) LTspice / Multisim MATLAB HTML / CSS

Projelerim

GİTT (Çip Teknoloji Projesi)

TÜBİTAK BİLGEM ve Aselsan için yürütülen AR-GE projesi. Visual Studio Code üzerinde VHDL ve FPGA Verilog kullanarak çip tasarımı geliştirdim. Teknofest'te finalist olduk.

VHDL FPGA Teknofest

MONARK Roket Projesi

Aviyonik sistem yazılımını Arduino ile geliştirdim. Kicad programında PCB kart tasarımı ve OpenRocket ile roket simülasyonları yaptım. Teknofest'te ÖTR raporunu başarıyla geçtik.

Arduino Kicad Aviyonik

TUGS (Uydu Görüntüleme Sistemi)

Türkiye'nin uydu görüntüleme sistemlerini geliştirmek amacıyla C++, Java, HTML ve CSS kullanarak arayüz ve sistem geliştirmesi yaptım.

C++ Java HTML/CSS

TBTT (Batarya Teknoloji Projesi)

TÜBİTAK TÜSSİDE için Li-İyon malzemeleri üzerine araştırma ve Kicad ile prototip devre tasarımı çalışmaları yürüttüm.
